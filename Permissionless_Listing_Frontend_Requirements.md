# Permissionless Listing - Frontend Requirements

## 1. Dashboard

| 區塊 | 顯示內容 |
|------|----------|
| 總覽 | 上線 Symbol 數、總交易量、總手續費分潤、總清算收益 |
| IF Account | 當前餘額、最低要求、狀態（正常/Warning/Limit） |
| Fee Account | 當前餘額、今日/本月/累計分潤、提領按鈕 |
| Liq Account | 當前餘額、最低要求、狀態 |
| 异常事件列表 | 時間、事件、狀態|
---

## 2. 帳號管理

### 2.1 IF Account
- 設置或更改按鈕
- 餘額總覽（當前餘額、最低要求、可提領金額）
- 充值功能
- 提領功能（含冷卻期倒數）
- 歷史紀錄（充值、提領、清算扣款）

### 2.2 Fee Account
- 設置或更改按鈕
- 餘額總覽（當前餘額、今日/本月/累計分潤）
- 提領功能
- 分潤明細（可按日期、Symbol 篩選、匯出）

### 2.3 Liquidation Account
- 設置或更改按鈕
- 餘額總覽（當前餘額、最低要求）
- 充值功能
- 清算執行紀錄

### 2.4 MM Account
- 帳號列表（名稱、綁定 Symbol、餘額、狀態）
- 新增/刪除/修改 MM Account
- 查看做市紀錄
- **MM Requirement 顯示**（見下方 Section 4）

---

## 3. Symbol 管理 (Symbol Management)

### 3.1 Symbol 列表
每個 Symbol 需顯示：
- 狀態（Pending / Active / Warning / Limit / Delisted）
- 今日交易量
- 今日手續費分潤
- 今日清算量 / 清算次數
- 當前流動性深度
- 操作按鈕（Reduce-only / Delist）

### 3.2 Symbol 詳情
- 上架參數查看/編輯
- 手續費查看/編輯

## 4. MM Requirement Dashboard

> **Note**: MM Requirement 的定義與顯示方式待確認

### 4.1 預計顯示內容

| 顯示項目 | 說明 |
|---------|------|
| 當前 Requirement | 各層級點差的深度要求 |
| 即時達成狀態 | 各層級是否達標（綠/紅燈） |
| Uptime 統計 | 今日/本週/本月的做市在線時間百分比 |
| 歷史紀錄 | 過去達成率趨勢圖 |

### 4.2 Requirement 表格範例

| Spread | Bid Req | Ask Req | Current | Status |
|--------|---------|---------|---------|--------|
| ±0.3% | $1,000 | $1,000 | $1,250 | ✓ 達標 |
| ±0.5% | $5,000 | $5,000 | $4,800 | ✗ 未達標 |
| ±1.0% | $25,000 | $25,000 | $28,000 | ✓ 達標 |

**狀態摘要：**
- Uptime 要求: 80%
- 今日 Uptime: 92%
- 狀態: 正常

## 5. 用戶端風險披露

### 5.1 Permissionless Symbol 標識UI

所有 Permissionless Listing 的 Symbol 需明確標識，例如：

```text
+-------------------------------------------------------------+
|  XYZ-PERP   [🏷️ Community Listed]                           |
+-------------------------------------------------------------+
|                                                             |
|  ⚠️ 風險提示                                                 |
|  此交易對由第三方項目方提供流動性與保險基金，風險較高。           |
|                                                             |
|                              [查看詳情]    [了解風險]        |
+-------------------------------------------------------------+
```
### 5.2 風險等級分類

| 風險等級 | 條件 | 顯示方式 |
|---------|------|---------|
| 低風險 | T1-T2 市值、IF > 200% 最低要求 | 綠色標籤 |
| 中風險 | T3 市值、IF 100-200% 最低要求 | 黃色標籤 |
| 高風險 | T4-T5 市值、IF < 120% 最低要求 | 紅色標籤 + 警示 |
